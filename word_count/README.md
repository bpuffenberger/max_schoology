# Word Count Exclusion Tool

A single-file GitHub Pages website that lets you:
- paste text into a large text area
- get a live word/token count
- optionally exclude numbers
- click words to exclude them from the adjusted total
- manually add excluded words
- copy a summary to the clipboard

## Files to upload
- `index.html`
- `.nojekyll` (optional but included)

## Quick GitHub Pages setup
1. Create a new GitHub repository.
2. Upload `index.html` and `.nojekyll` to the repository root.
3. In GitHub, open **Settings** > **Pages**.
4. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
5. Choose your branch (usually `main`) and the root folder (`/`).
6. Save.
7. Your site will publish at the GitHub Pages URL shown by GitHub.

## Local testing
Just open `index.html` in a browser.
